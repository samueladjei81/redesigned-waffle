there are 2 ways of opening and closing files:
1. Using the open() and close() tags 
2. using the 'with open' phrase
# the 'with open' phrase opens the file for use and closes it when the code is done instead of manually closing the file
#the open() and close() tags allow the programmer to manually open and close the file

1. USING THE open() and close() tags to implement readline() and readlines
#readline()
#open the file for reading
f1 = open('menu.txt')

#read line by line
ln = f1.readline()#read each line line by line but the readlines() reads all the lines in the file
while ln != '':
print(ln)
ln = f1.readline()

#close the file
f1.close()
